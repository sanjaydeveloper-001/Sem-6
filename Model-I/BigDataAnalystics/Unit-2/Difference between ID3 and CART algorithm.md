

# 14.b) Difference Between ID3 and CART Algorithm (20 Marks)

| Feature     | ID3                        | CART                               |
| ----------- | -------------------------- | ---------------------------------- |
| Full Name   | Iterative Dichotomiser 3   | Classification and Regression Tree |
| Type        | Classification only        | Classification and Regression      |
| Splitting   | Multi-way split            | Binary split                       |
| Metric Used | Information Gain (Entropy) | Gini Index                         |
| Tree Type   | Non-binary tree            | Binary tree                        |
| Data Type   | Mainly categorical         | Both numerical and categorical     |
| Pruning     | No pruning                 | Supports pruning                   |
| Complexity  | Simple algorithm           | More advanced algorithm            |
| Accuracy    | Moderate                   | Usually higher                     |

---

## ID3 Algorithm Characteristics

* Uses **entropy and information gain** to build decision tree. 
* Attribute with highest information gain becomes root node.

---

## CART Algorithm Characteristics

* Uses **Gini index** as splitting measure.
* Produces **binary decision trees**.
* Works for both **classification and regression tasks**. 

---

## Conclusion

ID3 and CART are both decision tree algorithms used in machine learning. ID3 mainly focuses on classification using entropy, while CART is more flexible as it supports both classification and regression with binary tree structures.
